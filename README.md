# Телеграм бот для управления заданиями

Простой телеграм бот для создания TODO списка с дедлайнами для сотрудников.

## Установка

1. Установите Python 3.8 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте бота в Telegram:
   - Найдите [@BotFather](https://t.me/BotFather) в Telegram
   - Отправьте команду `/newbot`
   - Следуйте инструкциям и получите токен бота

2. Создайте файл `.env`:
   - Скопируйте содержимое из примера ниже
   - Создайте файл `.env` в корне проекта
   - Вставьте ваш токен вместо `ваш_токен_здесь`

   ```
   BOT_TOKEN=ваш_токен_здесь
   ```

   **Важно:** Файл `.env` уже добавлен в `.gitignore`, чтобы токен не попал в git!

3. Запустите бота:
   ```bash
   python main.py
   ```

## Использование

После запуска бота, найдите его в Telegram и отправьте команду `/start`

### Доступные команды:

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/add_task` - Узнать, как добавить задание
- `/list_tasks` - Показать все задания
- `/complete_task [ID]` - Отметить задание как выполненное (например: `/complete_task 1`)
- `/delete_task [ID]` - Удалить задание (например: `/delete_task 1`)

### Как добавить задание:

Отправьте боту сообщение в следующем формате:
```
Задание: Подготовить отчет
Дедлайн: 25.12.2024
Сотрудник: Иван
```

Или просто:
```
Задание: Подготовить отчет
Дедлайн: 25.12.2024
```

## Хранение данных

Все задания сохраняются в файл `tasks.json` в той же папке, где находится бот.
